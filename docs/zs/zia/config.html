config¶

The following methods allow for interaction with the ZIA Activation Management API endpoints.

Methods are accessible via zia.config

class ActivationAPI(api)¶
activate()¶

Activates configuration changes.

Returns

str

Configuration status.

Examples

>>> config_activate = zia.config.activate()
status()¶

Returns the activation status for a configuration change.

Returns

str

Configuration status.

Examples

>>> config_status = zia.config.status()
